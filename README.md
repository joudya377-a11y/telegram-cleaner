# telegram-cleaner
​هذا المشروع عبارة عن بوت تلجرام مصمم لمسح الروابط تلقائياً من المجموعات للحفاظ على نظافة المحتوى. 
​🚀 كيف يعمل؟
​يقوم البوت بمراقبة الرسائل في المجموعة.
​إذا اكتشف رابطاً، يقوم بمسحه فوراً.
​🛠 طريقة التشغيل:
​للتشغيل، تأكدي من تثبيت المكتبات المطلوبة الموجودة في requirements.txt:
pip install -r requirements.txt
ثم شغل/ي الملف الرئيسي:
python main.py
🛡️ ملاحظة أمنية:
​يرجى عدم مشاركة الـ API Token الخاص بالبوت الخاص بك في أي ملف عام.
